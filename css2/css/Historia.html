Su historia


CSS fue propuesto primeramente por Håkon Wium Lie el 10 de octubre de 1994. Al mismo tiempo, Lie trabajaba con Tim Berners-Lee en el CERN. La primera Recomendación CSS por el W3C (CSS1)7 en 1996. En particular, la propuesta de Bert Bos fue influyente; el fue el coautor de CSS1y es reconocido cono el cocreador de CSS. CSS fue desarrollado para proveer hojas de estilos para la web. Un requerimiento para un lenguaje de hoja de estilo web era que las hojas de estilo vinieran en diferentes estilos en la web. CSS, permite al documento ser influenciado por múltiples hojas de estilo por medio de los estilos en «cascada». Mejorar las capacidades de la presentación en la web fue un tema de interés para muchos en las comunidades web, y nueve diferentes lenguajes de hojas de estilos fueron propuestas en la lista de correo www-style. De esas nueve propuestas, dos influenciaron profundamente en lo que sería CSS: Cascading HTML Style Sheets5 y Stream-based Style Sheet Proposal (SSP). 12 Dos navegadores fueron usados para pruebas para las propuestas iniciales; Lie trabajó con Yves Lafon para implementar CSS en el navegador Arena creado por Dave Raggett.13 14 15 Bert Bos implementó su propia propuesta SSP en el navegador Argo. Thereafter, Lie y Bos trabajaron juntos para desarrollar el estándar CSS. La 'H' se eliminó del nombre porque estas hojas de estilo pueden ser aplicadas a otros lenguajes de marcado además de HTML. La propuesta de Lie fue presentada en la conferencia "Mosaic and the Web" (mas tarde llamada WWW2) en Chicago, Illinois en 1994, y de nuevo con Bert Bos en 1995. En ese tiempo el W3C estaba siendo estabilizado, y mostraba interés en el desarrollo de CSS. En agosto de 1996 Netscape Communication Corporation presentó una alternativa de lenguaje de hoja de estilos llamada JavaScript Style Sheets (JSSS). La especificación nunca fue finalizada y ahora está obsoleta. A finales de 1996, CSS estaba listo para ser oficial, y la recomendación CSS 1 fue publicada en diciembre. El grupo de trabajo de CSS comenzó corrigiendo errores que no fueron anotados en el CSS 1, resultando en la creación de CSS 2 el 4 de Noviembre de 1997. Fue publicado como una recomendación el 12 de mayo de 1998. CSS 3, cuyo desarrollo inició en 1998, sigue en desarrollo en 2014. En 2005 el grupo de trabajo de CSS decidió mejorar los requerimentos de los estándares de una manera más estricta. Esto significó que a los estándares ya publicados como CSS 2.1, CSS 3 Selectors and CSS 3 Text pasaron de ser «Recomendaciones candidatas» a "Borrador de trabajo".
